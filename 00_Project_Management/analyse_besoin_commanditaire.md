# Analyse du besoin commanditaire

## Problématique
L'entreprise souhaite prédire les clients susceptibles de résilier afin d'anticiper les pertes et de mettre en place des actions de fidélisation.

## Objectifs
- Détecter les clients à risque.
- Comprendre les facteurs qui influencent le churn.
- Comparer plusieurs modèles de Machine Learning.
- Déployer le meilleur modèle dans une API.
- Transformer le score de churn en action métier.

## Contraintes
- Données tabulaires avec variables numériques et catégorielles.
- Cible potentiellement déséquilibrée.
- Besoin d'explicabilité.
- Besoin d'une solution maintenable et déployable.
